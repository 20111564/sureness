## <font color="green">Introduction</font>

> `sureness` 是面对`restful api`的鉴权`java`实现,在使用`shiro`过程中吸取其良好的设计加上一些自己想法实现的.  
其功能比较单一,基于`RABC`主要关注于对`restful api`的保护
